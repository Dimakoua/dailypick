How to Use Choices Spinner

Choices Spinner makes it easy to turn a custom list into a fair decision wheel. Start by editing the option list, then update the wheel and spin to get a result.

Step-by-step usage

  1. Edit the list: type your options in the left input area. Use one item per line or comma-separated values. Blank lines and duplicates are ignored.
  2. Update the wheel: press 🔄 Update after changing the list. This redraws the wheel using your current options and saves the list for your next visit.
  3. Spin the wheel: press 🎡 SPIN! to launch the spinner. The chosen option appears in the result box below.
  4. Reset anytime: press ↺ Reset to go back to the default options and redraw the wheel.

What each control does

  • Options input: Add, remove, or edit choices here to control what the spinner can choose from.
  • 🔄 Update: Saves your current list and redraws the wheel without spinning yet.
  • 🎡 SPIN! Spins the wheel and displays the selected item in the result area.
  • ↺ Reset: Restores the sample default list if you want to start fresh.

Why this tool works

This spinner keeps group decisions moving by removing the need for long debates. It’s ideal for lunch planning, meeting topics, random rewards, or any situation where a fair, unbiased pick helps the team move forward.

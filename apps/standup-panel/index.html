    Daily Stand-up Panel: Your All-in-One Stand-up Dashboard

    The Daily Stand-up Panel brings together everything you need to run a smooth daily stand-up meeting. Instead of juggling multiple tabs and tools, get a spinning wheel, countdown timer, speaker queue, and session stats in a single view.

    Built for scrum masters, engineering leads, and facilitators who want to keep stand-ups focused, fair, and on time. No sign-up required — everything runs in your browser.

    Why use the stand-up panel?

    • One screen: Wheel, timer, queue, and stats together — no tab switching.
    • Fair speaker selection: The wheel picks randomly so everyone gets a turn.
    • Built-in timing: Preset durations from 1 to 15 minutes, or set a custom time.
    • Session tracking: Streak counter, average duration, and recent speaker history.
    • Works with your roster: Reads the same player list from Settings → Roster.

    1. What is the Daily Stand-up Panel?

    The Daily Stand-up Panel is a unified dashboard that combines a speaker wheel, countdown timer, speaking queue, and session stats into a single page. It replaces the need to open separate tools for each part of your stand-up routine.

    2. How the stand-up panel works

    1. Set up your roster: Visit the Roster page to add your team members, or the panel will use a default list.
    2. Choose a game mode: Pick from six speaker-selection games — Wheel (spin to pick), Speedway (turtle race), Trap! (strategy), Falling Letters (pinball), Gravity Drift (space), or Patchinko (pachinko).
    3. Start the timer: Choose a preset duration or set a custom time, then hit Start.
    4. Track the queue: See who's speaking now, who's up next, and who's already gone.
    5. Review stats: Check your streak, session count, and average meeting duration.

    3. Key features

    • Six game modes: Wheel (spin to pick), Speedway (turtle race), Trap! (strategy), Falling Letters (pinball), Gravity Drift (space), and Patchinko (pachinko).
    • Integrated games: Wheel and Speedway run inline — no tab switching.
    • External games: Trap!, Falling Letters, Gravity Drift, and Patchinko open in a new tab and feed results back automatically.
    • Quick timer: Preset buttons (1m, 3m, 5m, 10m, 15m) plus custom input.
    • Speaker queue: Current, upcoming, and completed speakers at a glance.
    • Session stats: Streak tracking, total sessions, average duration, monthly count.
    • New Round button: Reset the queue and start fresh without reloading.
    • Persistent roster: Reads from the same localStorage roster used by all Daily Pick apps.
    • Game preference saved: Your chosen game mode is remembered across sessions.

    4. Use cases

    The stand-up panel is ideal for daily scrum meetings, team check-ins, sprint retrospectives, and any recurring meeting where you need to manage speaking order and time. It works great on a shared screen in the room or on a shared call.

    5. Tips for better stand-ups

    • Keep your roster updated so the wheel always reflects the current team.
    • Use the 5-minute preset for quick stand-ups, or 15 minutes for deeper syncs.
    • Click "New Round" at the start of each meeting to reset the queue.
    • Check the streak stat to build a consistent daily habit.

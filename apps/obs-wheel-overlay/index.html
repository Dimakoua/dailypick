📖 How to use OBS Wheel Overlay

This tool generates a transparent wheel URL you paste directly into OBS Studio, Streamlabs, XSplit, or any other software that supports Browser Sources. No account, no install, no plugin.

🟣 OBS Studio

  1. Configure your wheel above and click Generate Overlay URL.
  2. Copy the URL.
  3. In OBS, go to your scene → click ＋ in Sources → Browser Source.
  4. Paste the URL. Set Width and Height to match the wheel size you chose (add ~80px height for the title and spin button).
  5. Enable Shutdown source when not visible.
  6. Click OK, then right-click the source → Interact to spin from inside OBS without switching windows.

💡 The background is fully transparent — the checkered pattern in Preview is just to show transparency. OBS will show your game capture behind the wheel.

🟦 Streamlabs / SLOBS

  1. Generate the overlay URL above.
  2. In Streamlabs, go to your active scene → click ＋ → Browser Source.
  3. Paste the URL and set dimensions to match.
  4. Use the Interact option in the source context menu to click the spin button without alt-tabbing.

🎬 XSplit, vMix, Wirecast, Ecamm Live

All these tools support Browser Sources. Add a new Web Page / Browser source, paste the overlay URL, and match the dimensions. The background is transparent in all Chromium-based Browser Sources.

🔗 Sharing and updating your config

The full configuration — items, palette, size, spin duration, remove-winner setting — is encoded in the URL itself. To update your overlay mid-stream, edit the settings above, regenerate the URL, and refresh the Browser Source in OBS (right-click → Refresh).

Share the setup URL with a co-streamer and they can spin independently with an identical wheel configuration.

🖱 Spinning from OBS

Right-click the Browser Source in OBS and choose Interact. This opens a clickable preview window. Click SPIN! from there — no switching to a browser tab mid-stream.

💡 You can also bind the OBS source refresh to a hotkey using a browser-source-refresh script, so chat redemptions or alerts automatically trigger a spin by refreshing the page.

1. What is OBS Wheel Overlay?

OBS Wheel Overlay is a browser-based spin wheel with a transparent background, designed to sit on top of game capture, webcam, or any other OBS source. It requires no plugin, no sign-up, and no software beyond OBS itself. Configure the wheel on this page, copy the generated URL, and paste it into a Browser Source.

2. How it works

When the URL contains ?mode=overlay, the page strips all chrome (header, footer, instructions, setup panel), sets the background to transparent, and renders only the wheel, title, spin button, and result. OBS's Chromium-based Browser Source renderer supports transparent backgrounds natively when you enable the transparency option in the source settings.

3. Key features

  • Fully transparent background — no white or dark box behind the wheel, ever.
  • In-OBS spinning — use the Interact panel in OBS to click SPIN without changing windows.
  • Remove winner mode — each drawn item is removed from the wheel and can be restored.
  • URL-encoded config — items, palette, size, and settings live in the URL. Paste once, done.
  • Six streaming palettes — Neon, Twitch, YouTube, Sunset, Ocean, Forest.
  • Size control — 200 to 600px wheel diameter. Fit the wheel to your scene layout.
  • Live preview — see exactly how the overlay will look on a transparent background before copying the URL.

4. OBS Browser Source setup

In OBS Studio, add a Browser Source, paste the overlay URL, and set the width and height to match the wheel size you chose in the configurator. Enable "Shutdown source when not visible" to save GPU resources when the wheel scene is not active. Right-click the source and use Interact to access the spin button without alt-tabbing out of OBS.

5. Supported streaming platforms

The overlay works in OBS Studio, Streamlabs, Streamlabs OBS, XSplit Broadcaster, vMix, Wirecast, and Ecamm Live — any software that uses a Chromium-based Browser Source renderer. The streaming destination (Twitch, YouTube, Kick, Facebook Gaming, TikTok Live) does not affect the overlay at all.

6. Sharing the overlay URL

The configuration is fully encoded in the URL — no server, no database. Share the overlay URL with a co-streamer or moderator and they can open it in their own OBS Browser Source or browser tab to see and spin the same wheel. Removed winners are also encoded in the URL, so a shared URL mid-giveaway correctly shows which items remain.

7. OBS Wheel Overlay vs. Stream Wheel

The Stream Wheel is the full-featured multi-wheel tool for streamers — it includes a visible settings panel, multiple wheels, and an on-page configuration experience. OBS Wheel Overlay is the minimal counterpart: no on-screen controls in overlay mode, designed for streamers who configure once and just spin. Use Stream Wheel when you need to change items frequently on stream; use OBS Wheel Overlay when you want a clean, zero-UI overlay.

8. Alternatives

For a full-featured multi-wheel tool with on-screen settings, try the Stream Wheel. For a dedicated giveaway experience with tickets and multi-round draws, see the Stream Giveaway Picker. For a general-purpose decision wheel, the Choices Spinner works for non-streaming use cases.

    1. What Is the SuperEnalotto Number Generator?

    The SuperEnalotto Number Generator is a free, browser-based quick pick tool that creates random number combinations matching the official SuperEnalotto format: 6 main numbers from 1–90 and 1 Jolly number drawn independently from the same 1–90 pool. Every pick is statistically equivalent to quick picks generated by official lottery terminal machines in Italy.

    I built this because most online generators use Math.random(), which isn't cryptographically secure. This tool uses crypto.getRandomValues() with Fisher-Yates shuffling and rejection sampling to eliminate modulo bias. The Jolly number is drawn as a separate, independent event from the same pool — meaning it can match one of the main 6 numbers, exactly like the real game.

    3. What Are the SuperEnalotto Rules?

    SuperEnalotto uses a single-pool format with an independent bonus draw. Players pick 6 main numbers from a pool of 1–90. The Jolly number is drawn separately from the same 1–90 pool, meaning it can match one of the main numbers — this is a key difference from two-pool games like EuroJackpot or Powerball.

    • Main pool: 6 unique numbers from 1–90
    • Jolly: 1 number from the same 1–90 pool, drawn independently
    • Draw days: Tuesday, Thursday, and Saturday
    • Prize tiers: 6 total
    • Jackpot win: Match all 6 main numbers
    • 5+1 win: Match 5 main + Jolly (enhanced prize)
    • Minimum win: Match 2 main numbers (Tier 6)

    4. How Does the Number Generator Work?

    This generator uses crypto.getRandomValues() — the same cryptographic random number generator browsers use for security-sensitive operations — instead of Math.random(). For the main numbers, it builds a full pool of 1–90, applies a Fisher-Yates shuffle seeded by cryptographic randomness, and takes the first 6 values. The Jolly number is then drawn as a separate, independent random selection from the same 1–90 range. Rejection sampling eliminates modulo bias, and each line is an independent draw.

    1. What Is the EuroMillions Number Generator?

    The EuroMillions Number Generator is a free, browser-based quick pick tool that creates random number combinations matching the official EuroMillions format: 5 main numbers from 1–50 and 2 Lucky Stars from 1–12. Every pick is statistically equivalent to quick picks generated by official lottery terminal machines.

    I built this because most online generators use Math.random(), which isn't cryptographically secure. This tool uses crypto.getRandomValues() with Fisher-Yates shuffling and rejection sampling to eliminate modulo bias — ensuring every number has an equal chance of being selected.

    EuroMillions is drawn every Tuesday and Friday across 17 European countries. The jackpot starts at €17 million and can reach up to €250 million — making it Europe's biggest lottery prize.

    3. What Are the EuroMillions Rules?

    EuroMillions uses a two-pool format with two bonus numbers. Players pick 5 main numbers from a pool of 1–50 and 2 Lucky Stars from a separate, independent pool of 1–12. Both Lucky Stars are drawn from their own pool and are guaranteed to be different from each other.

    • Main pool: 5 unique numbers from 1–50
    • Bonus pool: 2 unique numbers from 1–12 (Lucky Stars, drawn separately)
    • Draw days: Tuesday and Friday
    • Prize tiers: 13 total
    • Jackpot win: Match all 5 main numbers + both Lucky Stars
    • Minimum win: Match 1 main number + 2 Lucky Stars

    4. How Does the Number Generator Work?

    This generator uses crypto.getRandomValues() — the same cryptographic random number generator browsers use for security-sensitive operations — instead of Math.random(). For the main numbers, it builds a full pool of 1–50, applies a Fisher-Yates shuffle seeded by cryptographic randomness, and takes the first 5 values. For the Lucky Stars, it shuffles a separate 1–12 pool and takes the first 2 values. Rejection sampling eliminates modulo bias, and each line is an independent draw.

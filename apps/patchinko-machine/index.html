1. What Is Patchinko Machine?

Patchinko Machine is a fair team order tool that drops balls through a field of pegs to determine stand-up sequence. The primary keyword is stand-up order picker, and the game makes participant selection more transparent and engaging.

The tool simulates a pachinko-style drop board so every team member can see how the queue forms as balls land into slots.

It is ideal for meetings, retros, and team activities where you want a shared, trustable method to choose who goes next.

2. How to Use It?

  1. Open Patchinko Machine and make sure your team roster is ready in Settings.
  2. Choose a peg pattern and click Drop Balls to release the order balls.
  1. Watch the balls navigate the pegs and land in the queue slots.
  1. Review the stand-up queue and use it to decide the next speaker or presenter.

3. Key Features & Customization

  • Fair visual randomization: everyone sees the same ball drop and slot result.
  • Multiple peg patterns: choose Grid, Tree, or Triangles for different queue behaviors.
  • Live queue display: the stand-up order updates instantly after each ball drop.
  • Settings integration: sync your roster and brand colors from the Settings page.
  • Easy reset: reset the board and rerun the drop when you need a new order.

4. Pachinko Queue Strategy

Patchinko Machine adds strategy to order selection by letting you choose the peg layout and then observing the ball path. Different patterns change how the balls navigate the board, making the queue generation feel more dynamic.

This visual process is useful for teams because it reinforces fairness—everyone can see that the order was determined by physics and random chance rather than a hidden hand.

The result is a more satisfying stand-up order that feels transparent and less arbitrary.

5. Sharing and Exporting

Patchinko Machine does not export the queue directly, but the current order is visible in the stand-up queue section and can be copied manually for meeting notes. Share the page link with your team if you want everyone to see the process live.

The tool is browser-based, so it is easy to use in remote or hybrid meetings without additional setup.

6. When to Use This Queue Tool?

Use Patchinko Machine when you want a visually engaging stand-up order process that feels fair and fun. It works especially well for teams that want to add a playful moment to meeting rituals.

  • When deciding stand-up speaker order in daily meetings.
  • For assigning demos or reports during sprint reviews.
  • When you want an impartial way to choose team volunteers.
  • During retro warmups to add a game-like element.
  • For remote teams who benefit from shared visual decision-making.

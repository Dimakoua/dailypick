🎡 Stream Wheel

A fully customizable spin wheel built for live streamers. Add multiple wheels on one page, configure each independently, remove winners round by round, and share the complete setup — or paste as an OBS Browser Source overlay.

📺 How to use Stream Wheel with OBS, Streamlabs & more

Stream Wheel runs entirely in the browser, no account required. Use it directly on this page, or paste as a transparent Browser Source overlay in any streaming software.

🚀 Getting started

  1. The wheel loads with example viewer names. Open ⚙️ Settings on any wheel card to change the items.
  2. Paste one name per line in the items box, then click 🔄 Update Wheel.
  3. Choose a color theme from the Color Theme palette row.
  4. Enable Remove winner from wheel after spin if you want each result removed automatically — ideal for giveaways.
  5. Click 🎡 SPIN! to spin. The result appears below the wheel.
  6. Removed winners appear as chips below the wheel. Click ✕ on any chip to restore that individual item, or use ↩ Restore All.
  7. Your full configuration (all wheels, results, history) is always encoded in the URL. Bookmark it or use the 🔗 Share Setup button.

🟣 OBS Studio

  1. Configure your wheels as desired — items, colors, titles.
  2. Click 📺 OBS Overlay URL above. This copies a URL with ?overlay=1 appended, which strips all menus and makes the background fully transparent.
  3. In OBS Studio, go to a Scene and click ＋ in the Sources panel.
  4. Select Browser Source and give it a name like "Stream Wheel".
  5. Paste the overlay URL into the URL field.
  6. Set Width to 450 and Height to 530 per wheel (scale up proportionally for multiple wheels).
  7. Enable Shutdown source when not visible to save resources.
  8. Click OK. Right-click the source and choose Interact to click the SPIN button directly from OBS without switching windows.

💡 Tip: Resize and reposition the Browser Source in your scene using the red handles. Hold Shift while resizing to scale freely.

🟦 Streamlabs / Streamlabs OBS

  1. Click 📺 OBS Overlay URL to copy your transparent overlay URL.
  2. In Streamlabs, go to your active Scene and click ＋ in the Sources section.
  3. Select Browser Source.
  4. Paste the overlay URL into the URL field.
  5. Set width to 450 and height to 530 (adjust for multiple wheels).
  6. Enable Shutdown source when not visible.
  7. Click Add Source. Use the Interact option in the source context menu to spin directly from Streamlabs.

🎬 XSplit Broadcaster

  1. Copy your overlay URL via the 📺 OBS Overlay URL button.
  2. In XSplit, go to your scene and click Add Source → Webpage.
  3. Paste the overlay URL and set the source dimensions to match your intended layout.
  4. The wheel loads with a transparent background and no extra UI.

📺 Twitch, YouTube Live & other platforms

Streaming platforms like Twitch and YouTube do not manage overlays — you add them in your broadcasting software (OBS, Streamlabs, XSplit). Follow the steps above to add Stream Wheel as a Browser Source, then stream to any platform as normal.

  • Twitch giveaways: enable "Remove winner after spin", add viewer usernames, and spin each round. Removed winners appear in the history chips.
  • YouTube milestones: create a reward wheel with prize names and spin live on screen for your viewers.
  • Discord communities: share the setup URL so community members see the exact same wheel configuration.
  • vMix, Wirecast, Ecamm Live: all support Browser Sources — follow the same URL + dimensions approach.

🔗 Sharing your configuration

Every wheel setting — titles, items, color themes, removed winners, spin duration — is encoded into the URL as you make changes. No account or save button required.

  • 🔗 Share Setup: copies a URL that opens this page with all your wheels pre-configured. Send it to a co-streamer or bookmark it.
  • 📺 OBS Overlay URL: same config but adds ?overlay=1. Paste directly into OBS / Streamlabs Browser Source. The page background becomes transparent and all menus hide automatically.
  • Auto-save: the URL updates silently as you configure without reloading the page. Your last configuration is always restorable from your browser history.
  • Sharing results: after a spin, removed winners are included in the URL, so sharing mid-giveaway lets your co-streamer resume exactly where you left off.

🔢 Using multiple wheels

Click ＋ Add Wheel to add more wheels to the page. All wheels are visible simultaneously — useful for showing a game wheel and a viewer wheel side by side on stream.

  • Each wheel has an independent items list, color theme, and remove-winner setting.
  • All wheels are included in the shared URL, so your entire multi-wheel layout is shareable as one link.
  • In overlay mode, all wheels render side by side with a transparent background.
  • Remove a wheel by opening its settings and clicking the ✕ button in the card header.

Stream Wheel — live stream spinner for Twitch, YouTube & OBS

Stream Wheel is a customizable spin wheel built specifically for live streamers. Unlike generic decision wheels, it supports multiple wheels on one page, removes winners round by round for fair giveaways, and outputs a transparent overlay URL you can paste directly into OBS Studio, Streamlabs, XSplit, or any Browser Source-compatible tool.

1. What is Stream Wheel?

Stream Wheel is a browser-based spin wheel designed for live content creators. It runs entirely client-side with no login, no ads blocking interaction, and no subscription. Configure one or more wheels, spin them on stream, and share the full setup through a single URL.

2. How it works

Open Stream Wheel, click the Settings icon on any wheel card, and paste your list of items — viewer names, game titles, challenge options, or prizes — one per line. Choose a color palette, set the spin duration, and hit SPIN. The wheel spins with a smooth ease-out animation and lands on a random item. If "remove winner" is enabled, the winner is removed from the wheel and shown in the history chips below. All this state is encoded in the URL automatically so nothing is lost between sessions.

3. Key features

  • Multiple wheels: Add as many wheels as needed. Each is fully independent with its own items, palette, and settings.
  • Remove winner mode: Each wheel can auto-remove the drawn item and track history. Restore any item individually or all at once.
  • OBS overlay mode: A single URL parameter strips all UI and makes the background transparent for use as an OBS Browser Source.
  • Shareable URL state: The complete configuration (titles, items, colors, history) is always in the URL — bookmark, share, or hand off mid-giveaway.
  • Six streaming-themed palettes: Neon, Twitch, YouTube, Sunset, Ocean, and Forest — or override with Brand Studio colors.
  • Adjustable spin duration: 2 to 12 seconds per wheel, independent per card.
  • Up to 200 items per wheel: For large viewer lists or competitions with many entries.

4. Color themes

Six pre-built palettes target common streaming aesthetics. Neon uses bright electric colors popular in gaming streams. Twitch uses the official Twitch purple and blue family. YouTube uses red, white, and the platform's accent colors. Sunset uses warm pinks and oranges for cozy or lifestyle streams. Ocean uses cool blues for a calm, professional look. Forest uses greens and earthy tones for nature or outdoor content.

5. Using it for giveaways

For a Twitch or YouTube giveaway, paste eligible viewer usernames into the items box. Enable "Remove winner after spin" so each drawn viewer is automatically removed from future spins. The winner list appears below the wheel in chips that your audience can see on screen. Share the overlay URL on Discord or with a co-streamer so they have live access to the same configuration.

6. OBS overlay mode

Adding ?overlay=1 to the URL activates overlay mode: header, footer, instructions, and configuration panels all disappear, and the page background becomes fully transparent. This lets the wheel render directly over your game capture or webcam in OBS without any white box or frame. You can interact with it using OBS's built-in "Interact" feature on the Browser Source, which opens a clickable preview — no alt-tabbing required.

7. Sharing and URL state

Stream Wheel uses base64-encoded JSON in the URL to store all state. No server, no database, no cookies. Every wheel title, every item, every removed winner, and every color choice is encoded and updated silently as you make changes. The URL is always shareable. If you want to send someone your exact giveaway setup mid-stream, click Share Setup and paste the link into Discord. They open it and see your wheels in exactly the same state.

8. When to use Stream Wheel

  • Viewer giveaways and prize draws on Twitch or YouTube Live
  • Randomizing which game to play next on stream
  • Challenge wheels, truth-or-dare prompts, or viewer-submitted ideas
  • Team splits and random matchup generators for community tournaments
  • Viewer-engagement moments: "spin the wheel" segments that keep chat engaged
  • Subscription milestone rewards and channel point redemptions

9. Alternatives

If you need a general-purpose decision wheel without streaming features, try the Choices Spinner. For a minimal transparent overlay without configuration UI, the OBS Wheel Overlay is purpose-built for Browser Sources.

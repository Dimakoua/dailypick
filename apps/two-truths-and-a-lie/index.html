Host remote icebreakers with confidence

Two Truths and a Lie is ideal for meetings, onboarding, and remote team standups. This live room puts the host in control while the group makes guesses in real time.

How it works

  • Host sets the round: Add three statements and select which one is the lie.
  • Team votes: Everyone chooses the statement they believe is false.
  • Reveal together: The host shares the screen and reveals the lie once the round is complete.

Why this format helps

Live rooms keep everyone focused on the same screen, while the host manages timing and reveal. It’s a great way to learn fun facts about teammates and spark conversation quickly.

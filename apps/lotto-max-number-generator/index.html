    1. What Is the Lotto Max Number Generator?

    The Lotto Max Number Generator is a free, browser-based quick pick tool that creates random number combinations matching the official Lotto Max format: 7 main numbers from 1–50 and 1 bonus number drawn from the remaining pool. Every pick is statistically equivalent to quick picks generated by official lottery terminal machines.

    I built this because most online generators use Math.random(), which isn't cryptographically secure. This tool uses crypto.getRandomValues() with Fisher-Yates shuffling and rejection sampling to eliminate modulo bias — ensuring every number has an equal chance of being selected.

    Lotto Max is Canada's biggest lottery, drawn every Tuesday and Friday. The jackpot starts at $10 million and can grow to $70 million, with MaxMillions prizes adding extra $1-million draws when the pot exceeds $50 million.

    3. What Are the Lotto Max Rules?

    Lotto Max uses a same-pool bonus format. Players pick 7 main numbers from a pool of 1–50. After the 7 main numbers are drawn, a bonus number is drawn from the remaining 43 numbers — guaranteeing it is different from all main numbers. This is a key difference from two-pool lotteries like Powerball or Mega Millions.

    • Main pool: 7 unique numbers from 1–50
    • Bonus: 1 number from the remaining 43 (same pool, guaranteed distinct)
    • Draw days: Tuesday and Friday
    • Prize tiers: 7 total
    • Jackpot win: Match all 7 main numbers
    • Second prize: Match 6 main numbers + bonus

    4. How Does the Number Generator Work?

    This generator uses crypto.getRandomValues() — the same cryptographic random number generator browsers use for security-sensitive operations — instead of Math.random(). It builds a full pool of 1–50, applies a Fisher-Yates shuffle seeded by cryptographic randomness, and takes the first 7 values as main numbers. The bonus number is then drawn from the remaining 43 values. Rejection sampling eliminates modulo bias, and each line is an independent draw.

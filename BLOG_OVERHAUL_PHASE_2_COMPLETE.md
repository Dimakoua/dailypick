## Lessons Learned (Phase 1 → Phase 2)

1. **Hub-and-spoke model works:** One comprehensive post (hub) + multiple supporting posts (spokes) = content gravity. Users dig deeper.
2. **Target keywords first:** Each pillar post targets specific high-volume queries, then builds sideways to related keywords.
3. **Integration >> Replacement:** Daily Pick tips enhance game formats; they don't replace them. Users read game content, then discover Daily Pick naturally.
4. **FAQ sections drive featured snippets:** Each FAQ in Phase 2 posts targets "People Also Ask" box queries.
5. **Internal linking forces UX improvement:** Linking old to new content surfaces weak posts and forces quality improvements.
